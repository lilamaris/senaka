# AGENTS.md

이 문서는 구현 단계에서 에이전트/개발자가 필요한 문서를 빠르게 찾기 위한 라우팅 엔트리입니다.

## 문서 라우팅 순서
1. 아키텍처 개요: `docs/ARCHITECTURE.md`
2. 기능별 모듈 문서: `docs/modules/*.md`
3. 실행 TODO: `docs/todo/*.md`

## 빠른 링크
- 아키텍처 루트: `docs/ARCHITECTURE.md`
- TODO 인덱스: `docs/todo/README.md`

## 구현 시 권장 탐색 흐름
1. `docs/ARCHITECTURE.md`에서 전체 구성과 기능 루트 파악
2. 해당 기능의 `docs/modules/<기능>.md`에서 모듈 구성/흐름/사용법 확인
3. `docs/todo`에서 목적별 작업 항목 실행
